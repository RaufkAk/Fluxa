# Fluxa E-Ticaret Projesi

Bu proje, modern bir e-ticaret deneyimi sunmak için geliştirilmiş, HTML, CSS ve JavaScript ile yazılmış bir web uygulamasıdır.

## Özellikler
- Ana sayfa, ürünler, hizmetler, hakkında, sepet ve istek listesi sayfaları
- Ürünleri sepete ve istek listesine ekleme
- Sepet ve istek listesi verilerinin localStorage ile saklanması
- Sepet ve istek listesi arayüzünde canlı güncelleme
- Responsive (mobil uyumlu) tasarım

## Kurulum & Çalıştırma
1. Projeyi klonlayın:
   ```
   git clone https://github.com/RaufkAk/Fluxa.git
   ```
2. Klasöre girin ve `index.html` dosyasını bir tarayıcıda açın.
